Getting Started

Moon Animator 2 exposes its internal API through a global table, allowing plugin developers to extend functionality, create custom tools, and automate animation workflows.

Unofficial Documentation
This API is not officially documented by Moon Animator developers. Structure may change between versions. Use at your own risk.

Access Point

All Moon Animator internals are accessible through the global table:

Lua 
local _g = _G.MoonGlobal

-- Wait for Moon Animator to fully load
repeat
    _g = _G.MoonGlobal
    task.wait(0.5)
until _g and _g.ready

print("Moon Animator version:", _g.ver) -- 34000

Easing Functions

Easing functions control animation interpolation. Each function takes a progress value t (0-1) and returns the eased value (0-1).

Lua 
local _g = _G.MoonGlobal
local progress = 0.5

-- Get eased value
local eased = _g.EasingFunctions.QuadOut(progress)
print(eased) -- 0.75

Direction Types

DirectionDescriptionCurve
In Starts slow, accelerates ⟋ (ease in)
Out Starts fast, decelerates ⟍ (ease out)
InOut Slow start and end, fast middle S-curve
OutIn Fast start and end, slow middle Inverse S-curve

Themer

The Themer handles all color/theme management for Moon Animator's UI.

Key Properties

PropertyTypeDescription
_Theme table Current theme colors (Color3 values)
_BuiltInThemes table All built-in themes (RGB arrays: {R, G, B})
_PaintMap table Property → theme key mapping
_PaintedItemMap table Painted UI elements
theme_buttons table Theme selection buttons

Key Methods

MethodDescription
:SetTheme(name) Apply a theme by name
:_paintAll() Repaint all UI elements with current theme

Built-in Themes

Free Themes
Default, Dark, Light, and others included with Moon Animator
Paid Themes
Taiga, Blush, Blonde, Wheat, Gaia, Ares, Aqua, Energy, Digital, Fuchsia
Theme Color Format
Built-in themes store colors as {R, G, B} arrays (0-255 range).
Runtime _Theme uses Color3 objects. Convert with Color3.fromRGB().
Lua 
local _g = _G.MoonGlobal
local T = _g.Themer

-- Get current theme color
local mainColor = T._Theme.main
print(mainColor) -- Color3 value

-- Get built-in theme data (RGB arrays)
local taigaTheme = T._BuiltInThemes["Taiga"]
for key, rgb in pairs(taigaTheme) do
    local color = Color3.fromRGB(rgb[1], rgb[2], rgb[3])
    print(key, color)
end

-- Apply a theme
T:SetTheme("Dark")
T:_paintAll()

Plugin Modding

Examples of extending Moon Animator 2 with custom functionality. Save these as Local Plugins in Roblox Studio (right-click script → "Save as Local Plugin...").

Tip: Waiting for Moon Animator
Always wait for _G.MoonGlobal to be ready before accessing the API. Use a repeat loop with task.wait() to avoid errors.

Adding Custom Easing Functions

This example adds two custom easing types (Soft and Tert) with full UI integration:

Lua — CustomEasings.lua 
-- CustomEasings — Add custom easing functions to Moon Animator 2
-- Save as Local Plugin

local CUSTOM_EASINGS = {
    {
        name        = "Soft",
        displayName = "Soft",
        color       = Color3.fromRGB(86, 180, 233),
        power       = 1.5,
    },
    {
        name        = "Tert",
        displayName = "Tert",
        color       = Color3.fromRGB(230, 159, 0),
        power       = 2.5,
    },
}

-- Easing function generators
local function makeIn(p)
    return function(t) return t^p end
end

local function makeOut(p)
    return function(t) return 1 - (1 - t)^p end
end

local function makeInOut(p)
    return function(t)
        if t < 0.5 then
            return ((2 * t)^p) / 2
        else
            return 1 - ((-2 * t + 2)^p) / 2
        end
    end
end

local function makeOutIn(p)
    return function(t)
        if t < 0.5 then
            return (1 - (1 - 2 * t)^p) / 2
        else
            return ((2 * t - 1)^p + 1) / 2
        end
    end
end

-- Wait for Moon Animator
local _g
for i = 1, 60 do
    _g = _G.MoonGlobal
    if _g and _g.EasingFunctions and _g.class then break end
    if i == 60 then return end
    task.wait(0.5)
end

local Ease = require(_g.class.Ease)

-- Register each custom easing
for _, easing in ipairs(CUSTOM_EASINGS) do
    local name = easing.name
    local p = easing.power

    -- Add easing functions
    _g.EasingFunctions[name .. "In"]    = makeIn(p)
    _g.EasingFunctions[name .. "Out"]   = makeOut(p)
    _g.EasingFunctions[name .. "InOut"] = makeInOut(p)
    _g.EasingFunctions[name .. "OutIn"] = makeOutIn(p)

    -- Add to Ease data for UI
    Ease.EASE_DATA[name] = {
        Color = easing.color,
        Params = { Direction = true }
    }

    -- Add to ease list
    table.insert(Ease.EASE_LIST, { name, easing.displayName })
end

print("[CustomEasings] Loaded", #CUSTOM_EASINGS, "custom easings")

Theme Unlocker

This example unlocks all paid themes for local use:

Lua — ThemeUnlocker.lua 
-- ThemeUnlocker — Unlock all paid themes in Moon Animator 2
-- Save as Local Plugin

local _g
for i = 1, 60 do
    _g = _G.MoonGlobal
    if _g and _g.Themer and _g.AllMenuItems then break end
    if i == 60 then return end
    task.wait(0.5)
end
task.wait(1)

local T = _g.Themer

local PAID_THEMES = {
    "Taiga", "Blush", "Blonde", "Wheat", "Gaia",
    "Ares", "Aqua", "Energy", "Digital", "Fuchsia"
}

-- Apply theme from built-in data
local function applyTheme(themeName)
    local src = T._BuiltInThemes[themeName]
    if not src then return false end

    -- Convert {R, G, B} → Color3 and write to _Theme
    for key, value in pairs(src) do
        if typeof(value) == "table"
            and typeof(value[1]) == "number"
            and typeof(value[2]) == "number"
            and typeof(value[3]) == "number"
        then
            rawset(T._Theme, key, Color3.fromRGB(
                value[1], value[2], value[3]
            ))
        end
    end

    -- Repaint UI
    pcall(function()
        T:_paintAll()
    end)

    -- Save preference
    pcall(function()
        _g.plugin:SetSetting(_g.theme_key, themeName)
    end)

    return true
end

-- Unlock paid themes: hide price, enable click
for _, themeName in ipairs(PAID_THEMES) do
    for _, prefix in ipairs({"Theme_", "Activate_Theme_"}) do
        local item = _g.AllMenuItems[prefix .. themeName]
        if not item then continue end

        -- Hide price tag
        if item.price_frame then
            item.price_frame.Visible = false
        end

        -- Connect click handler
        local input = item.middle_ui and item.middle_ui:FindFirstChild("Input")
        if input and input:IsA("GuiButton") then
            input.MouseButton1Click:Connect(function()
                applyTheme(themeName)
            end)
        end
    end
end

print("[ThemeUnlocker] Unlocked", #PAID_THEMES, "themes")

Tips for Plugin Development

TipDescription
Wait for Ready Always wait for _g.ready == true before accessing UI elements
Save Settings Use _g.plugin:SetSetting(key, value) and :GetSetting(key)
Window Access Most windows have .UI property pointing to their main Frame
Input Elements Windows often have .g_e containing their input/control elements
Menu Items Access via _g.AllMenuItems["ItemName"], click via .middle_ui.Input
Error Handling Wrap API calls in pcall() to prevent plugin crashes
